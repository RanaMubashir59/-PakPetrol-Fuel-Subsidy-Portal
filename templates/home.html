Platform features
Everything you need in one portal

CNIC-based identity

Unique 13-digit CNIC format validation ensures no duplicate registrations across the entire system.

Secure QR redemption

One-time tokens expire in 10 minutes. Cryptographically signed UUIDs that cannot be reused or forged.

Real-time balance

Balance deducted instantly when fuel is dispensed. Both rider and station see live confirmation.

Live analytics

City-wise transaction data, station performance, and monthly credit summaries for administrators.

Dynamic petrol pricing

Admin updates OGRA petrol prices in real time. All redemptions calculate automatically at current rates.

Fraud detection

Automatic flagging of rapid redemptions, multi-city same-day usage, and duplicate token attempts.

Fraud detection rules

Rapid redemption check

Flags if same rider attempts redemption within 30 minutes of a previous transaction

Multi-city detection

Alerts if transactions occur in two different cities on the same calendar day

QR token reuse

Tokens are one-time use only — any reuse attempt is immediately blocked and logged

Balance guard

Server-side validation prevents dispensing more fuel than the available credit balance
